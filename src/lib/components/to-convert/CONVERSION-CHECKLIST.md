## Conversion Process

### For Each Template:

1. **Extract HTML**
   - Open the source `index.html` file
   - Locate the component code blocks
   - Extract the HTML structure

2. **Convert to Svelte**
   - Convert HTML to Svelte syntax
   - Replace Tailwind classes with shadcn-svelte components where applicable
   - Add Svelte props and reactivity

3. **Create Component File**
   - Create `.svelte` file in appropriate directory
   - Add TypeScript types
   - Export component

4. **Register in Template Registry**
   - Add to `template-registry.ts`
   - Add type definitions in `types.ts`
   - Export from `index.ts`

5. **Test Component**
   - Verify rendering
   - Test responsiveness
   - Check dark mode support

6. **Document**
   - Add usage examples
   - Document props
   - Add to documentation
