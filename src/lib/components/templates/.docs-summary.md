### Single Source of Truth Pattern

```typescript
// 1. Define in template-registry.ts
defaultProps: {
  title: 'Start your journey with',
  brandName: 'Mindhyv',
  reviews: [...]
}

// 2. Component pulls from registry

let {
  title = DEFAULTS.title,
  reviews = DEFAULTS.reviews
}: Props = $props();
```
